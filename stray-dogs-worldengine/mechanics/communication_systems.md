# Communication Systems

## Overview
How the **Stray Dogs** coordinate operations securely.

## Methods
1.  **Encrypted Apps**: Signal/WhatsApp (used by 65 members).
    *   **High Command Chat**: "The Pack" (Big 9, Dame, Snout, Zed, Chew-toy).
    *   **Frontline Chat**: "NEPH Dogs" (Ruckus, Matches, Whistle, Needles).
    *   **Ghost Chat**: "Eyes" (Snout, SEPH scouts).
2.  **Burner Phones**: Disposable, for critical ops or anonymous tips.
3.  **Visual Signals**: Red shoe laces, specific graffiti tags (Zed's tally marks).
4.  **Whistle**: Immediate distress signal (Whistle carries one).
5.  **SD Media**: Uses music videos to broadcast philosophy ("Muzzle Up, Teeth Down").

## Security
-   **No Names**: Use code names (Stray, Big 9, Dame, etc.) in chats.
-   **Delete Messages**: Auto-delete enabled.
-   **Silence Protocol**: No verbal communication with police.

## Significance
-   **Speed**: Enables rapid response (Level 3/4 threats).
-   **Survival**: Protects identities from police/rivals.
