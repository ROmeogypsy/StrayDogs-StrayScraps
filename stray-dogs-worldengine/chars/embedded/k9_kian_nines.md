# K-9 / Kian Nines (The Voice)

-   **Build**: Lean, wiry, restless energy.
-   **Face**: Face tattoos (small, tasteful). Intense eyes.
-   **Clothing**: Academy navy blazer (worn loose), heavy chains, streetwear.
